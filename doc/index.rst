Welcome to vhsven-sklearn's documentation!
==========================================

This project is an extension to scikit-learn's tree algorithms. 
In particular, a :class:`pruneabletree.prune.PruneableDecisionTreeClassifier`
is added. It supports two well-known pruning methods:

- Reduced Error Pruning
- Error Based Pruning
